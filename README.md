# **DocAgent-mini: AI‑агент для работы с документацией**
**версия 2.0**

**DocAgent-mini** — это учебный мини‑проект AI‑агента, который отвечает на вопросы пользователей по внутренней документации компании через HTTP‑API. Проект объединяет ключевые концепции современной разработки AI‑решений: RAG (Retrieval‑Augmented Generation), управление состоянием (stateful agents), безопасность и структурированное логирование.

## **Цель проекта версии 2.0**

Создать готовый к деполю AI‑агент, который:

-   ищет информацию в локальной документации (Markdown‑файлы);
-   создаёт и обновляет задачи (тикеты);
-   запоминает состояние для каждого пользователя между запросами;
-   предоставляет простой HTTP‑интерфейс для взаимодействия;
-   соответствует базовым стандартам продакшен‑кода (логирование, обработка ошибок, валидация).
    

Проект предназначен для закрепления навыков работы с LLM, RAG и stateful‑агентами, а также понимания принципов работы технологий langchain и langgraph. В версии 2.0 отрабатываются подходы к созданию полноценного, готового к продакщену приложения. Его можно использовать как стартовую точку для создания корпоративного AI‑помощника.

## Технологический стек
-   **Python 3.10+**  — основной язык разработки;
- **fastapi 0.136** — фреймворк для создания HTTP‑API;
-  **Uvicorn**  — ASGI‑сервер для запуска приложения;
-  **pytest**  — тестирование;

## Установка
1.  Клонируйте репозиторий:
```
    git clone <https://github.com/YuriiGaponov/DocAgent-mini.git>
    cd DocAgent-mini
   ```

2.  Создайте виртуальное окружение:
```
   python -m venv venv
   ```
4.  Активируйте виртуальное окружение:
        -   **Windows:**
        ```
        venv\Scripts\activate
        ```
        -   **macOS/Linux:**
        ```
        source venv/bin/activate
        ```
5.  Установите зависимости:
   ```
    pip install -r requirements.txt
   ```

## Запуск
Запустите сервер с помощью Uvicorn: uvicorn main:app --reload Сервер будет доступен по адресу:  `http://localhost:8000`

## Документация API (Swagger/OpenAPI)

FastAPI автоматически генерирует интерактивную документацию API в формате Swagger/OpenAPI. Она доступна по стандартному эндпоинту:

-   **URL:** `http://localhost:8000/docs`

## Запуск тестов

Выполните команду ```pytest``` в корне проекта.
